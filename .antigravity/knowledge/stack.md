# 🧱 Tech Stack

## Frontend Core

- **React 19** – Core library for building user interfaces
- **React DOM** – DOM rendering for React applications
- **React Router v7** – Client-side routing and navigation

## State Management & Data Fetching

- **TanStack React Query** – Server-state management, caching, and async data handling

## Forms & Validation

- **React Hook Form** – Performant form handling
- **Zod** – Schema validation and type-safe data parsing
- **@hookform/resolvers** – Integration between React Hook Form and Zod

## Styling & UI

- **Tailwind CSS v4** – Utility-first CSS framework
- **@tailwindcss/vite** – Tailwind integration with Vite
- **tailwind-merge** – Utility class merging
- **clsx** – Conditional className handling
- **class-variance-authority (CVA)** – Component variant management
- **@base-ui/react** – Headless UI components

## UX & Utilities

- **Sonner** – Toast notifications
- **nuqs** – URL query state management

## Build Tools

- **Vite** – Fast build tool and dev server
- **@vitejs/plugin-react** – React support for Vite
- **vite-plugin-svgr** – Import SVGs as React components

## Code Quality & Linting

- **ESLint** – Linting and code quality
- **@eslint/js** – ESLint base config
- **eslint-plugin-react-hooks** – Enforces React Hooks rules
- **eslint-plugin-react-refresh** – Ensures safe Fast Refresh
- **globals** – Global variables configuration

## Type Support

- **@types/react** – Type definitions for React
- **@types/react-dom** – Type definitions for React DOM
